## Installation

Run the following command in a terminal and you're ready to go!

**Windows**
```powershell
iwr instl.sh/srobroek/facc/windows | iex
```

**macOS**
```bash
curl -sSL instl.sh/srobroek/facc/macos | bash
```

**Linux**
```bash
curl -sSL instl.sh/srobroek/facc/linux | bash
```
